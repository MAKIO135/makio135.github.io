# **CV**

### Web Projects:
- **SharismLab**: Data Visualisation / Twitter API ( `NodeJS` / `p5js` )
- **SuperFeedr**: Real time DataViz of web trendings topics, to approach new clients: Acquired by Medium ( `NodeJS` / `Redux` / `React` / `D3` )
- **BimViewer**: Manipulation and visualization of BIM models ( `Redux` / `React` / `ThreeJS` )
- **Topogram**: Visualizing and editing networks data collaboratively ( `Meteor` / `React` / `cytoscape` / `Materialize` )
- **Drawn Engine**: Generating machine code + computing prices + exporting JSON files for parametric 3D printed objects using `Grasshopper` + `ghpython` and visualizing them in a commercial website using `Wordpress` and `ThreeJS`
- **LeClub.io**: Sharing graphic and interactive web experiments ( `github pages` / `Jekyll` / `Sass` / `D3` / `p5js` / `ThreeJS` )
- More on `openprocessing.org` / `Codepen.io`

#### Tools:
- **MorePVector**: a Processing library allowing to use PVector in drawing functions ( Java )
- **PVectorjs**: a JavaScript 2D / 3D vector library with chainable methods
- **SensorShieldLib**: an Arduino library to easily watch and communicate sensors values in JSON ( C++, used by Cheval Vert for dataphyz workshops )
- **console.chart**: a javascript library for visualizing arrays as bar charts in the console
